# 🎹 Python Keylogger (Educational)

This project is a basic **keylogger** written in Python using the `pynput` library. It logs all keystrokes to a local file and is designed for **ethical and educational** purposes only.

---

## ⚙️ Features

- Logs keystrokes in real-time
- Saves to a secure local file
- Lightweight and simple

---

## 🔐 Ethical Disclaimer

> ⚠️ **This tool is created for educational use only.**  
> Do not use this keylogger on any system without explicit permission.

---

## 🛠️ Technologies Used

- Python 3
- `pynput` library
